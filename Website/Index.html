Welcome to my to-do-list website
There are two tables which can be used, they are tasks and people with the following functions:

- Read All (returns all data set)
- Read by (returns a specific data set - required an ID - Currently Not Working)
- Create (creates a new data set)
- Update (updates a data set - requires an ID)
- Delete (delete a data set - requires an ID)

Please navigate to the Tasks or People pages on the navigation bar to perform the functions.
